async puppet.connection.gen_broadcasts()[source]¶

Asynchronously listen for and yield valid broadcast packets.

Listens for UDP packets on the broadcast port. Yields decoded JSON objects and their source addresses for packets that start with the expected magic number.

Yield:

Tuple of (info, addr), where info is the decoded JSON object and addr is the sender’s address.

Return type:

AsyncGenerator[Tuple[dict, Tuple[str, int]], None]

Core constants and enums for the Puppeteer communication system.
